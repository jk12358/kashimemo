# Claude AI 開発ガイドライン

## 開発環境

- **OS**: Windows
- **エディタ**: VSCode
- **ターミナル**: PowerShell

## 基本ルール

### 最近・直近など、現在日時を取り扱う場合
- 必ず `date` コマンドを使って、現在日時を取得すること

### コミュニケーション
- **回答言語**: 全て日本語で行う

### バージョン管理
- **コード管理**: GitHub
- **重要**: Claude AIはGitHubの操作を行わない
- Git関連の操作（commit, push, pull等）はユーザーが実行する

### 開発方針
1. **段階的な進行**
   - 一度に全てを変更しない
   - 小さな変更を積み重ねる

2. **タスク管理**
   - 複数のタスクを同時並行で進めない
   - 1つのタスクを完了してから次へ進む

3. **エラー対応**
   - エラーは必ず解決してから次のステップへ進む
   - エラーを無視して次に進むことは厳禁

4. **機能追加**
   - 必要な機能は事前に確認してから実装する

## 注意事項
- ユーザーは自分でコードを書くことが頻繁にあり、このことをしっかりと留意しておくこと。基本的にコードを書いたり変更したりしたらその都度伝える。
- 各ステップで確実に動作を確認すること
- 問題が発生したら、その場で解決すること
- 不明点があれば、推測せずに確認すること
